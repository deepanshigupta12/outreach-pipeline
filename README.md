# Outreach Pipeline Automation

## Overview

This is a CLI-based outreach automation system built using Node.js.

It takes a company domain as input, fetches contacts, and sends personalized emails using Brevo API.

---

##  Features

* Domain-based input
* Fetch contacts (mock / API-ready)
* Confirmation before sending emails
* Personalized email sending
* Modular architecture

---

##  Tech Stack

* Node.js
* Axios
* Brevo API

---

## ▶ How to Run

```bash
npm install
node index.js amazon.com
```
